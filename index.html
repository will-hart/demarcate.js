demarcate.js

demarcate.js is a jQuery based editor for Markdown. It lets you edit directly in a page and generate Markdown from the HTML elements. demarcate.js is under development by William Hart for the online publishing service to-textr. It supports many basic Markdown functions, however support for extensions is still limited. The library is offered under the GPLv3 license

Where to get started

Integrating into your site

Editing is not currently supported on mobile browsers and is untested in Internet Explorer

Integrating into your website only requires a few lines of code. The editor uses your site's CSS styles for styling text, however the editor itself requires a small CSS stylesheet. Include this by putting <link rel="stylesheet" href="css/demarcate.css"> somewhere in your head.

Next you need to include jQuery and the demarcate.min.js file (which includes showdown) somewhere near the end of your <body> section and initialise a DOM object as the editor container. 

<script src="http://code.jquery.com/jquery.min.js" type="text/javascript"></script>
<script src="js/demarcate.min.js" type="text/javascript"></script>
<script>
    $(document).ready(function(){
        $("#preview_markdown").click( function() {
            alert(demarkdown($("#container")));
        });
    });
</script>
